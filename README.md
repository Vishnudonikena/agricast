# 🌾 AgriCast – Crop Price Prediction System

## 📌 Overview

AgriCast is a data-driven crop price prediction system that helps farmers and stakeholders make informed decisions by forecasting future crop prices using historical data and machine learning models.

This project focuses on predicting agricultural commodity prices (e.g., wheat) to reduce uncertainty and improve planning in the farming ecosystem.

---

## 🚀 Features

* 📊 Predicts future crop prices using ML algorithms
* 📈 Analyzes historical agricultural data
* ⚡ Simple and interactive user interface
* 🧠 Data preprocessing and feature engineering
* 📉 Helps farmers make better selling decisions

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/Vishnudonikena/agricast.git
cd agricast
```

### 2️⃣ Create virtual environment

```
python -m venv .venv
```

### 3️⃣ Activate environment

```
.venv\Scripts\activate   # Windows
```

### 4️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 5️⃣ Run the project

```
python app.py
```
